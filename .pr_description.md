## Summary
- Add a `pr-review-toolkit` skill for findings-first PR reviews against explicit diffs instead of broad codebase skims.
- Add bundled review references and a `collect_pr_context.py` helper to summarize commits, hotspots, file categories, and test signals before deeper review.
- Add a `talk-it-through` skill for chat-only explanation and brainstorming requests that should stay read-only.

## Changes
- **PR review skill package**: Add `skills/pr-review-toolkit/SKILL.md`, `skills/pr-review-toolkit/agents/openai.yaml`, and reviewer guidance in `skills/pr-review-toolkit/references/`.
- **Review context helper**: Add `skills/pr-review-toolkit/scripts/collect_pr_context.py` to inspect range, staged, or working-tree diffs and emit markdown or JSON review context.
- **Discussion skill package**: Add `skills/talk-it-through/SKILL.md` and `skills/talk-it-through/agents/openai.yaml` to support advisory, no-edit conversations.
